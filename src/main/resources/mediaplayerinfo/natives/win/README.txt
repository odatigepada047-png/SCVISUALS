ВАЖНО: Для работы музыки в Dynamic Island нужна нативная библиотека MediaPlayerInfo.dll

Эта библиотека взаимодействует с Windows Media Session API для получения информации о проигрываемой музыке.

УСТАНОВКА:

1. Скачайте MediaPlayerInfo.dll из одного из источников:
   - Оригинальный репозиторий (если найдете)
   - Скомпилируйте из исходников C++

2. Положите файл MediaPlayerInfo.dll в эту папку:
   src/main/resources/mediaplayerinfo/natives/win/MediaPlayerInfo.dll

3. Пересоберите проект: ./gradlew build

АЛЬТЕРНАТИВА:

Если не можете найти DLL, музыка просто не будет работать, но остальной функционал мода будет работать нормально.

Код автоматически обработает отсутствие библиотеки и не будет крашить игру.
